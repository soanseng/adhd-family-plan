# ADHD 家庭本位計劃指南

這份指南給台灣家長、老師與臨床工作者使用。它不是診斷工具，而是一套把日常照顧變得更容易執行的家庭工作手冊。

適合使用：

- 想把早晨、作業、睡前或情緒衝突變得更可預測的家庭。
- 想用簡單表格與老師溝通的家長。
- 想把家長訓練、生活結構、正向增強轉成可執行步驟的助人工作者。

不適合只靠本工具：

- 孩子有自傷、攻擊、明顯憂鬱、強烈焦慮、嚴重睡眠問題、學習退化、疑似自閉症或其他發展困難。
- 家庭正在處理高度衝突、暴力、照顧者耗竭或安全風險。
- 正在使用藥物但想自行停藥或調整劑量。

> 本網站提供 ADHD 家庭支持與教育資訊，不能取代醫師、心理師、職能治療師、特教老師或其他專業人員的評估與治療。若孩子出現明顯情緒低落、強烈焦慮、攻擊行為、嚴重睡眠問題、學習退化、疑似自閉症或其他發展困難，請尋求專業協助。若孩子正在使用藥物，請勿自行停藥或調整劑量，請與醫師討論。

## 使用方式

1. 先用產生器產生 4 週或 8 週計劃。
2. 只選一個最痛點開始。
3. 列印對應表格貼在家中可見位置。
4. 每週回顧一次：保留有效策略，刪掉太難的要求。
5. 有安全、發展、學習或情緒疑慮時，帶著紀錄尋求專業協助。
